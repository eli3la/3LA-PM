# 3LA PM

Internal project management dashboard.

## Features
- Hierarchical management (Category → Project → Sub-project → Tasks)
- Full CRUD editing at all levels (inline rename, add, delete)
- People tagging on projects, sub-projects, and tasks
- Notes & attachments on projects, sub-projects, and tasks
- Reminders with person-filtered email digest
- Visual Gantt timeline with category filtering
- Export any view as formatted text (copy or email)
- Team management (add, edit, delete members)

## Deploy

```bash
npm install
npm run build
```

Deploy the `dist/` folder to Vercel, Netlify, or any static host.

## Dev

```bash
npm install
npm run dev
```
